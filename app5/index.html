Instrucciones de Carga

Para que los suministros se muestren correctamente en el mapa, el archivo Excel debe contener las siguientes columnas:

  • SUMINISTRO: (Obligatorio) ID para ubicar en la matriz.
  • ACTIVIDAD: (Opcional) A1, A2, A3, NO ASIGNADO, IMPOSIBILIDAD.
  • FRENTE: (Opcional) Para agrupar y filtrar por frentes de trabajo.
  • VER EN CAMPO / REUBICACION: (Opcional) Marcar con "SI".
  • DISTRITO / LOCALIDAD: (Opcional) Para filtros geográficos.
  • USUARIO / MEDIDOR / DIRECCION: (Opcional) Datos informativos.
  • CATEGORIA / OBSERVACION: (Opcional) Detalles adicionales.

Nota: Si la palabra "CAMPO" aparece en cualquier columna, el punto se marcará automáticamente en color naranja.
