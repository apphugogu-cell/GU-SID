Instrucciones de Carga

Para que los suministros se muestren correctamente en el mapa, el archivo Excel debe contener las siguientes columnas:

  • SUMINISTRO: (Obligatorio) Número identificador para ubicar en la matriz.
  • ACTIVIDAD: (Opcional) Define el color (A1, A2, A3, PROYECTO, IMPOSIBILIDAD, OTROS).
  • VER EN CAMPO: (Opcional) Si se pone "SI", el punto será naranja.
  • REUBICACION: (Opcional) Si se pone "SI", el punto será morado.
  • DISTRITO: (Opcional) Para usar los filtros del panel lateral.
  • USUARIO / MEDIDOR: (Opcional) Para mostrar en el panel de detalles.

Nota: Si la palabra "CAMPO" aparece en cualquier columna, el punto se marcará automáticamente en color naranja.
